Accessibility Statement

Our commitment to an inclusive experience

KamiKoi Sushi Fusion is committed to ensuring our website is accessible to people of all abilities. We aim to conform with the Web Content Accessibility Guidelines (WCAG) 2.1 Level AA.

What We’ve Done

This site is built with semantic HTML, keyboard-navigable menus, visible focus states, sufficient color contrast, descriptive alt text and an accessibility toolbar offering options such as larger text, high contrast and reduced motion.

Ongoing Effort

Accessibility is an ongoing effort and we continue to improve. If you encounter any barrier using this site, please let us know so we can help and make it right.
